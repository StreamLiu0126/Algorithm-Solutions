# P1435 [IOI 2000] 回文字串

[P1435 [IOI2000] 回文词](https://www.luogu.com.cn/problem/P1435)

## 1. 核心思路：从递归开始

这道题的思路其实很直观，我们先不考虑优化，用**递归**的思想来拆解问题。

对于一个字符串，如果我们想让它变成回文串，可以使用两个指针 $l$ 和 $r$ 分别指向首和尾：

1. **字符相等**：如果 $s[l] == s[r]$，说明两头已经对称了，不需要任何操作。直接向内收缩，处理 $[l+1, r-1]$ 区间。
2. **字符不等**：此时我们需要进行插入操作。有两种选择：
   - 在 $l$ 左侧补一个 $s[r]$：此时新补的字符与 $s[r]$ 匹配，剩下的区间是 $[l, r-1]$。
   - 在 $r$ 右侧补一个 $s[l]$：此时新补的字符与 $s[l]$ 匹配，剩下的区间是 $[l+1, r]$。
   - **取两者最小值**，并记作一次操作。

> **思考**：为什么不用在两边各自补充一个字符？
>
> 因为我们只需要一次操作就可以消除两端的不对称性（让其中一个字符找到匹配），没必要同时补两个。

### 递归代码实现 (f1)

C++

```c++
int f1(int l, int r) {
    // 基础情况：区间只有一个字符，本身就是回文
    if (l == r) return 0;
    // 基础情况：区间有两个字符
    if (l + 1 == r) {
        return (s[l] == s[r] ? 0 : 1);
    }

    if (s[l] == s[r]) {
        return f1(l + 1, r - 1);
    } else {
        return min(f1(l + 1, r) + 1, f1(l, r - 1) + 1);
    }
}
```

## 2. 记忆化搜索：加上缓存表

为了避免重复计算，我们引入一个二维数组 `dp[l][r]` 来记录已经计算过的结果。

### 记忆化代码实现 (f2)

C++

```c++
int f2(int l, int r) {
    if (dp[l][r] != -1) return dp[l][r]; // 命中缓存

    if (l == r) {
        dp[l][r] = 0;
        return 0;
    }
    if (l + 1 == r) {
        dp[l][r] = (s[l] == s[r] ? 0 : 1);
        return dp[l][r];
    }

    if (s[l] == s[r]) {
        dp[l][r] = f2(l + 1, r - 1);
    } else {
        dp[l][r] = min(f2(l + 1, r) + 1, f2(l, r - 1) + 1);
    }
    return dp[l][r];
}
```

## 3. 严格位置依赖：迭代式 DP

通过转移方程发现，`dp[l][r]` 的值取决于三个位置：

1. **左边**：`dp[l][r-1]`
2. **下边**：`dp[l+1][r]`
3. **左下**：`dp[l+1][r-1]`

### 填表逻辑

以长度为 6 的字符串为例，我们需要的是 `dp[0][5]`。

- **排除区域**：对角线左下方（$l > r$）的位置可以忽略。

- **对角线**：$l == r$ 时，只有一个字符，值为 0。

- **次对角线**：$l + 1 == r$ 时，两个字符相等为 0，不等为 1。

- **遍历顺序**：为了保证计算当前格时，左、下、左下都已经确定，我们采用**从底到顶（i 递减）**，**从左到右（j 递增）**的顺序。

  ![image-20260422133757748](C:\Users\22767\AppData\Roaming\Typora\typora-user-images\image-20260422133757748.png)

### 迭代代码实现 (f3)

C++

```c++
void f3() {
    // 初始化对角线
    for (int i = 0; i < n; i++) {
        dp[i][i] = 0;
    }

    // 从底向上填表
    for (int i = n - 1; i >= 0; i--) {
        for (int j = i + 1; j < n; j++) {
            // 处理长度为 2 的特例
            if (j == i + 1) {
                dp[i][j] = (s[i] == s[i + 1] ? 0 : 1);
                continue;
            }

            if (s[i] == s[j]) {
                dp[i][j] = dp[i + 1][j - 1]; // 依赖左下角
            } else {
                dp[i][j] = min(dp[i + 1][j], dp[i][j - 1]) + 1; // 依赖下边和左边
            }
        }
    }
    cout << dp[0][n - 1];
}
```

## 4. 终极优化：空间压缩

既然 `dp[i]` 只与“当前行”和“下一行”有关，我们可以将二维数组压缩为一维。

使用变量 `pre` 来暂存被覆盖前的旧值，它模拟的就是原本的**左下角**元素。

![image-20260422133742680](C:\Users\22767\AppData\Roaming\Typora\typora-user-images\image-20260422133742680.png)

### 空间压缩实现 (f4)

C++

```
void f4() {
    vector<int> dp1(n, 0); 
    
    for (int i = n - 1; i >= 0; i--) {
        int pre = 0; // 初始 pre 代表 dp[i+1][j-1] 在 j=i+1 时的情况
        for (int j = i + 1; j < n; j++) {
            int tmp = dp1[j]; // 记录下当前位置的值，它将作为下一个 j 的“左下角”
            
            if (s[i] == s[j]) {
                // 如果相等，直接等于左下角
                if (j == i + 1) dp1[j] = 0;
                else dp1[j] = pre; 
            } else {
                // 如果不等，等于 min(下方旧值, 左方新值) + 1
                dp1[j] = min(dp1[j], dp1[j - 1]) + 1;
            }
            pre = tmp; // 更新 pre
        }
    }
    cout << dp1[n - 1];
}
```
